回望支持

如果你在使用回望时遇到问题，可以先查看下面的常见问题。

数据保存在哪里？

当前版本默认将目标、每日记录和复盘数据保存在设备本地，不需要账号。

换手机前应该怎么备份？

请在 App 的“数据”页生成 JSON 备份，并保存到你信任的位置。换机后可以通过“导入 JSON 备份”恢复。

购买 Pro 后没有解锁怎么办？

请进入“数据”页点击“恢复购买”。如果仍未恢复，请确认你使用的是购买时的 Apple ID。

如何取消订阅？

请在 iOS 的 App Store 账户订阅管理页面取消订阅。取消后，Pro 权益通常会保留到当前已付款周期结束。

联系我们

请通过 GitHub Issues 提交问题或功能建议。提交问题时，建议说明设备型号、iOS 版本、App 版本和复现步骤。

如果问题涉及个人隐私，请不要在公开 Issue 中粘贴每日记录、目标内容、备份文件或其他敏感信息。
