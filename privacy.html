回望隐私政策

生效日期：2026-07-09

回望是一款本地优先的目标复盘 App。我们重视你的个人记录和目标数据，因此当前版本默认不需要账号，也不会将你的目标、每日记录、复盘内容或备份文件上传到我们的服务器。

我们收集哪些数据

当前版本不收集可用于识别你的个人数据。

你在 App 内创建或填写的长期目标、季度目标、本周目标、每日三问记录、周复盘、月复盘、JSON 备份和 Markdown 导出内容，默认保存在你的设备本地。

本地数据存储

回望会把 App 数据保存在设备本地的应用沙盒中。你可以在 App 内生成 JSON 备份或 Markdown 导出文件，并自行选择是否保存、分享或迁移这些文件。

如果你删除 App，设备本地保存的数据可能会一并删除。建议在换机或删除 App 前先导出 JSON 备份。

App 内购买

回望 Pro 的购买、订阅、退款和账单处理由 Apple 的 App Store 和 StoreKit 提供。回望只读取 Apple 返回的购买状态，用于判断是否解锁 Pro 功能。

我们不会接触或保存你的银行卡号、账单地址、Apple ID 密码等支付信息。

数据共享

当前版本不会把你的目标、记录、复盘或备份文件共享给第三方。如果你主动使用系统分享功能导出 Markdown 或 JSON 文件，文件会交由你选择的系统分享目标处理。

追踪和广告

当前版本不包含第三方广告，不进行跨 App 或跨网站追踪。

政策变更

如果未来版本增加账号、云同步、AI 分析或其他在线服务，我们会在上线前更新本隐私政策，并在 App Store Connect 中同步更新隐私信息。

联系方式

如果你对隐私政策有疑问，可以通过 支持页面 联系我们。
