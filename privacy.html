回望隐私政策

更新日期：2026-07-17

回望是一款本地优先的目标复盘 App。我们重视你的个人记录和目标数据，因此当前版本默认不需要账号，也不会将你的目标、每日记录、复盘内容或备份文件上传到我们的服务器。

我们收集哪些数据

当前版本不收集可用于识别你的个人数据。

你在 App 内创建或填写的长期目标、季度目标、本周目标、每日三问记录、周复盘、月复盘、JSON 备份和 Markdown 导出内容，默认保存在你的设备本地。

本地数据存储

回望会把 App 数据保存在设备本地的应用沙盒中。你可以在 App 内生成 JSON 备份或 Markdown 导出文件，并自行选择是否保存、分享或迁移这些文件。

如果你删除 App，设备本地保存的数据可能会一并删除。建议在换机或删除 App 前先导出 JSON 备份。

App 内购买

回望 iOS 版的购买、订阅、退款和账单处理由 Apple App Store 和 StoreKit 提供；HarmonyOS 版由华为应用市场和华为 IAP Kit 提供。

回望只读取当前平台返回的商品和有效购买状态，用于判断是否解锁 Pro 功能。我们不会接触或保存你的银行卡号、支付密码或平台账号密码。

网络权限

HarmonyOS 版会使用网络访问权限连接华为 IAP Kit，用于查询商品、完成购买和恢复购买。回望不会使用该权限上传你的目标、记录或备份文件。

数据共享

当前版本不会把你的目标、记录、复盘或备份文件共享给第三方。如果你主动使用系统文件选择器或分享功能导出 Markdown 或 JSON 文件，文件会交由你选择的保存位置或分享目标处理。

追踪和广告

当前版本不包含第三方广告，不进行跨 App 或跨网站追踪。

政策变更

如果未来版本增加账号、云同步、AI 分析或其他在线服务，我们会在上线前更新本隐私政策，并在相应的应用商店后台同步更新隐私信息。

联系方式

如果你对隐私政策有疑问，可以通过 支持页面 联系我们。
